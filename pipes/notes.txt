Built-in Pipes in Angular
Currency Pipe - CurrencyPipe is a built-in pipe in Angular that is used to format a number as a currency value.
Date Pipe
Json Pipe
LowerCase Pipe
UpperCase Pipe
PercentPipe - PercentPipe is a built-in pipe in Angular that is used to transform a number into a percent value.
SlicePipe - SlicePipe is a built-in pipe in Angular that is used to create a new array or string that contains a portion of an existing array or string.
TitleCasePipe
AsyncPipe - AsyncPipe is a built-in pipe in Angular that is used to handle asynchronous data streams. It is commonly used to subscribe to observables or promises and display the emitted values in the view.


Chaining in Pipes:-
    Chaining pipes in Angular involves applying multiple pipes in sequence to transform data in a template. 
    You can chain pipes by using the pipe operator (|) multiple times, with each pipe representing a separate transformation.
